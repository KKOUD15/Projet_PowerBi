# Dashboard Industriel Power BI 🏭📊

**Conception d’un tableau de bord décisionnel pour le pilotage de la performance industrielle.**

## 🎯 Objectif du projet
Transformer des données industrielles brutes en indicateurs décisionnels clairs et exploitables. Ce projet vise à offrir une vue globale et détaillée sur les opérations de production pour faciliter la prise de décision.

## ⚙️ Indicateurs clés (KPIs) intégrés
- **Suivi de production :** Analyse des jobs terminés, en retard ou échoués.
- **Fiabilité opérationnelle :** Taux de disponibilité et de performance des machines.
- **Optimisation énergétique :** Suivi de la consommation d'énergie par type d'opération.
- **Analyse des goulots d'étranglement :** Identification des causes de retards.

## 🛠️ Outils utilisés
- **Microsoft Power BI** (Modélisation de données, formules DAX, conception de l'interface)
- **Analyse de données industrielles**

*Projet académique réalisé durant l'année universitaire 2025-2026 (Filière GMSI N7).*
